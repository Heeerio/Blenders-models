# Blenders-models/2D Graphics

1. Latarnia
Model latarni wykonany podczas praktyk zawodowych jako asset środowiskowy do gry komputerowej.
Proces obejmował stworzenie geometrii high poly, następnie retopologię do wersji low poly, bevelowanie krawędzi. Model został zoptymalizowany pod wykorzystanie w silniku gry i dostosowany do stylistyki projektu.

2. Książka
Model książki wykonany na potrzeby projektu uczelnianego.
Obiekt zaprojektowano jako interaktywny element otoczenia, z naciskiem na czytelną sylwetkę, poprawne proporcje oraz niską liczbę polygonów.

3. Klucz
Model klucza wykonany w ramach projektu uczelnianego jako przedmiot kolekcjonerski i element mechaniki gry.
Skupiono się na prostocie siatki, czytelnej formie oraz poprawnej skali. Model został zoptymalizowany pod real-time rendering i łatwą implementację w silniku gry.

4. Latarka
Model latarki przygotowany na potrzeby projektu uczelnianego.
Obiekt zaprojektowano jako funkcjonalny rekwizyt gry, z uwzględnieniem podziału na logiczne elementy konstrukcyjne. Model przystosowany do animacji oraz wykorzystania efektów świetlnych w silniku gry.

5. Potwór skalny
Model potwora skalnego wykonany w ramach projektu uczelnianego.
Projekt skupiał się na stworzeniu masywnej, czytelnej sylwetki oraz charakterystycznej formy wizualnej. Model stanowi bazę pod dalsze etapy pracy, takie jak teksturowanie, rigging lub animacja.

 Grafiki 2D – Dywany do gry
Wzory dywanów (3 warianty kolorystyczne)

Zestaw graficznych wzorów dywanów wykonanych podczas praktyk zawodowych na potrzeby gry komputerowej.
Dla każdego wzoru przygotowano trzy warianty kolorystyczne, zgodne z wytycznymi stylistycznymi projektu.

Prace obejmowały:

projekt wzoru dywanu,

dobór palety barw,

wielowarstwową koloryzację,

dopasowanie estetyki do klimatu gry,

przygotowanie grafik do implementacji jako tekstury.

Dywany pełnią funkcję dekoracyjną w środowisku gry, wzbogacając jej wizualną spójność i atmosferę.
